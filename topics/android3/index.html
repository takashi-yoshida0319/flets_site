2015年3月19日

セキュリティ対策ツール　for Androidのバージョンアップに関するお知らせ

セキュリティ対策ツール　for Android 3.2　を提供開始いたします。

平素よりNTT西日本のフレッツサービスをご利用いただきまして、誠にありがとうございます。
「フレッツ 光ネクスト」および「フレッツ 光ライト」（いずれもインターネット接続サービス）をご利用のお客さまへ、 Android端末向けとして提供しております「セキュリティ対策ツール　for Android」の最新版（セキュリティ対策ツール　for Android 3.2）を提供開始します。
なお、最新版の特徴は下記のとおりです。

最新版の特徴

「プライバシースキャン機能」がさらに充実

  1. 個人として許容するアプリについては、対象アプリを承認済みリストに追加することで、検索の対象から外すことができます。
    （プライバシースキャン機能は、個人情報漏えいの可能性があるアプリをチェックし警告を表示します。）

NTT西公式アプリへの導線追加

  1. ホーム画面のメニューからNTT西公式アプリへ接続できるようになりました。

＜バージョンアップ開始日＞

平成27年3月19日（木）

＜対象となるお客さま＞

「フレッツ 光ネクスト」および「フレッツ 光ライト」をご利用の方で、「セキュリティ対策ツール　for Android 1.0/2.0/3.0/3.1」を下記OSにてご利用中の方

Android OS 2.3 / 3.0 / 3.1 / 3.2 / 4.0 / 4.4 / 5.0

＜バージョンアップ手順＞

「セキュリティ対策ツール　for Android」の最新版（セキュリティ対策ツール for Android3.2）のバージョンアップを実施してください。

・ 光BOX＋でご利用中の方
こちらの手順をごらんください。
・ Android端末でご利用中の方
こちらの手順をごらんください。

＜バージョン確認方法＞

メイン画面下部の「セキュリティ対策ツール　for Android X.0」をご確認ください。
インストール済みのバージョンが確認できます。
