セキュリティ対策ツールのMac対応について

2014年11月10日

NTT西日本では、「フレッツ 光ネクスト」、「フレッツ 光ライト」（いずれもインターネット接続サービス） でパソコン（Windows OS）、スマートフォン、タブレット（Android OS） 等の機器をご利用のお客さま向けに提供している「セキュリティ 対策ツール」 ※1 について、平成26年12月22日より、新たにMac パソコンに対応した「セキュリティ対 策ツール for Mac」を提供開始いたします。
本提供により、これまでのWindows OS及びAndroid OSに加え、Mac OS ※2 搭載機器でのセキュリ ティー対策を行うことが可能となります。

※1  トレンドマイクロ株式会社の技術を採用しています。
※2  Mac OS X 10.8/10.9/10.10 に対応予定です。

1. サービス概要

（1） 「セキュリティ対策ツール for Mac」の主な機能

コンピューターウイルス対策／
スパイウェア対策		 ファイルの読み込みや書き込みを監視し、
ウイルスやスパイウェアに感染したファイルの駆除や隔離処理を実行します。
Web 脅威対策／
有害サイト規制 ※3		 フィッシングやワンクリック詐欺等の危険なWebサイトや、
有害な情報を含むサイトへのアクセスを制限します。
個人情報保護 ※3 対応するSNS のプライバシー設定内容（情報公開の範囲等）を確認し、
よりプライバシー情報を安全に保つ設定を行えるようにします。
※3  トレンドマイクロ株式会社が提供する機能です。
【「セキュリティ対策ツール for Mac」の機能について】
 ・  ご利用の端末環境（OSバージョン等）やソフト等との相性により本機能が正常に動作しない場合があります。
 ・  定義ファイルやプログラムは、更新を行い、常に最新の状態にしていただく必要があります。
 ・  本機能はセキュリティーに対する全ての脅威への対応を保証するものではありません。
