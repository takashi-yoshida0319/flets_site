2014年3月25日

セキュリティ対策ツール　for Androidのバージョンアップに関するお知らせ

セキュリティ対策ツール　for Android 3.0を提供開始いたします。

平素よりNTT西日本のフレッツサービスをご利用いただきまして、誠にありがとうございます。
「フレッツ 光ネクスト」および「フレッツ 光ライト」（いずれもインターネット接続サービス）をご利用のお客さまへ、Android端末向けとして提供しております。「セキュリティ対策ツール　for Android」の最新版（セキュリティ対策ツール　for Android 3.0）を提供開始します。
なお、最新版の特徴は下記のとおりです。

最新版の特徴

「不正アプリ対策」の機能追加☆

（1）クラウドからの検索を追加しました。クラウドに問い合わせすることで最新のセキュリティ対策を実現します。

（2）SDカード内の検索も可能です。

（3）検索対象のファイルを「すべてのファイル」または「APKファイルのみ」のいずれかを選択できます。

☆ 一部機能はトレンドマイクロ(株)が提供します。

「プライバシースキャン」の機能追加★

（1）アプリケーションインストール時に、個人情報漏えいの可能性をチェックし警告を表示します。

（2）インストール済のアプリケーションも、手動スキャンできます。

（3）検索対象のファイルを「すべてのファイル」または「APKファイルのみ」のいずれかを選択できます。

★ トレンドマイクロ(株)が提供します。

＜バージョンアップ開始日＞

平成26年3月25日（火）

＜対象となるお客さま＞

「フレッツ 光ネクスト」および「フレッツ 光ライト」をご利用の方で、「セキュリティ対策ツール　for Android 1.0/1.1/2.0」を下記OSにてご利用中の方
Android OS 2.2 / 2.3 / 3.0 / 3.1 / 3.2 / 4.0/ 4.1/ 4.2/ 4.3

＜バージョンアップ手順＞

「セキュリティ対策ツール　for Android」の最新版（セキュリティ対策ツールFor Android3.0 ）のバージョンアップを実施してください。

・ 光BOX＋でご利用中の方
    こちらの手順をごらんください。

・ Android端末でご利用中の方
    こちらの手順をごらんください。

＜バージョン確認方法＞

メイン画面下部の「セキュリティ対策ツール　for Android X.0」をご確認ください。
インストール済みのバージョンが確認できます。
