マイクロソフト社におけるWindows XPのサポート終了に伴うお知らせ

2014年2月10日

平素よりNTT西日本のフレッツサービスをご利用いただきまして、ありがとうございます。

平成26年4月9日をもってマイクロソフト社によるWindows XPのサポート終了が予定されておりますので、同日をもってフレッツサービス※1のサポートについても対象外とさせていただきます。

現在、Windows XPでフレッツサービスをご利用のお客さまにつきましては、サポート終了後もフレッツサービスをご利用いただくことができますが、マイクロソフト社から新たなセキュリティファイルや修正モジュールの提供が行われないことにより不具合が発生した場合、弊社として対応致しかねることがございます。
（弊社の「セキュリティ対策ツール」につきましては、多くのお客さまからご要望をいただいたこともあり、サポート終了後もWindows XP環境において当面はご利用いただくことができますが、セキュリティ脅威から十分にお客さま端末を守ることが出来ない可能性がございます。）

Windows XPをご利用のお客さまについては、上記の点に十分ご留意の上、最新のOSへの移行を強くお勧めします。
なお、OS移行をご検討のお客さまにつきましては、新しいOSの利用方法等についてサポートを行う「リモートサポートサービス」等のサポートサービスを弊社でもご用意しておりますので、併せてご検討下さい。

NTT西日本では、上記の「リモートサポートサービス」に加え、2台目以降のパソコンまたはスマートフォン・タブレット端末でセキュリティー機能を利用できる「セキュリティ機能ライセンス・プラス」、PC等の自然故障を補償する「ひかり機器保証」をまとめた「フレッツあんしんパック」をお勧めしております。詳しくは下記URLをご確認下さい。
