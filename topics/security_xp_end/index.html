「セキュリティ対策ツール」のWindows XPのサポート終了について

2015年12月22日

平素よりNTT西日本のフレッツサービスをご利用いただきまして、ありがとうございます。

平成26年4月9日をもってマイクロソフト社によるWindows XPのサポートが終了し、それに伴いフレッツサービス※1のサポートについても対象外とさせていただきましたが、一部「セキュリティ対策ツール（Ver.6 / Ver.21）」についてはサポート終了後もご利用いただくことができておりました。

しかし、平成27年12月28日をもって、「セキュリティ対策ツール（Ver.6 / Ver.21）※2」でのWindows XPのサポートを終了させていただきます。

「セキュリティ対策ツール」での動作要件の詳細はこちらをご確認ください。

※1  「フレッツ 光ネクスト」「フレッツ 光ライト」「フレッツ・光プレミアム」「Bフレッツ」「フレッツADSL」「フレッツ・スポット」「フレッツ・v6アプリ」「フレッツ・プラス」「フレッツ・セッションプラス」「フレッツ・ｖ6オプション」「ひかり電話」に関するサービス等
※2  「セキュリティ対策ツール（Ver.8 / Ver.23）」 は、Windows XPにはもともと対応しておりません。
