ランプ状況の診断手順(詳細)

  1. 診断結果を確認する

    診断結果

    ホームゲートウェイ部分（またはルーター部分）の異常により、インターネットが繋がらなくなっていると考えられます。

    下記へお進みください。

    「PPP」が消灯している

    「PPP」が消灯している

    このランプで機器がインターネットに接続しているかどうかがわかります。消灯の場合は機器類の設定確認を行いましょう。

    機器類の設定確認はこちら

    以下の場合「PPP」は正常です。

    「PPP」が緑点灯している。

    ISP（プロバイダー：インターネット接続事業者）の設定が正しく入っていれば、インターネットにつながります。緑点灯で正常の状態です。

    「PPP」がオレンジ点灯している。

    同時に2箇所の接続が機器で行われている時にはランプがオレンジに点灯します。
    どちらかの設定にISP（プロバイダー：インターネット接続事業者）の設定が正しく入っていれば、インターネットにつながります。オレンジ点灯で正常の状態です。
