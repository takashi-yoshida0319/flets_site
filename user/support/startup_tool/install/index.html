スタートアップツール インストール方法

動画で見る「スタートアップツール」

インストール方法について

「スタートアップツール」のインストールは「フレッツ簡単セットアップツール」を使って行います。
※お客さまの環境により、インストール手順や画面が一部異なる場合がありますので、ご了承ください。

「フレッツ簡単セットアップツール」をダウンロードし、デスクトップ上に保存した[FLETS_setup_W.exe]をダブルクリックで実行します。

【ご注意】
ご利用の環境によっては、ファイルの拡張子(.exe)は表示されない場合があります。

設定内容の選択画面で、「アプリケーションインストール」をクリックします。

「スタートアップツール」にチェックを入れ、「次へ」をクリックします。

「次へ」をクリックします。

以上で設定は完了です。
「完了」をクリックして終了してください。

  • ＊「フレッツ・v6アプリ」をご利用の場合、「簡単セットアップツール」からインストールしてください。
  • ＊ダウンロードできない場合は、ボタン上で右クリックし、「名前を付けてリンク先を保存」を選択のうえファイルをダウンロードしてください。
  • ＊対象OSについてはこちらをご確認ください。
